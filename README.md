**Verifiable algorithmic edge settling on decentralized ledgers.**

Brier Protocol is the infrastructure layer for algorithmic talent. We allow quantitative researchers and developers to deploy prediction bots, prove their mathematical edge immutably (Shadow Phase), and eventually unlock non-custodial capital via ERC-4626 Vaults. 

Builders earn a share of the profit generated by their algorithms, never risking their own capital.

> Ranked by math, not marketing. Predicts on Polymarket. Settled on Polygon.

## 1. The Core Vision: Data as the Ultimate Moat

Brier is not a casino nor a simple DeFi application. It is a **Verifiable Decentralized Oracle of Predictions**. 
Our core asset is the **Event Bus**: an append-only, cryptographically signed dataset of "who knew what, with what certainty, and when". 

Every prediction made by a bot is signed via HMAC and timestamped **before** the market resolves. This creates an immutable track record free from survivorship bias and cherry-picking.

## 2. Architecture & The SDK-First Approach

Brier is designed to be interacted with via our SDKs (Python & TypeScript), acting as the bridge between off-chain heavy Machine Learning models and on-chain settlement.

```text
Builder ──(HMAC Signed Signal)──► Brier Executor (Ingest API)
                                      │
  (Shadow Phase)                      ├─► Event Bus (Immutable Log)
  Resolves against Polymarket CLOB ───┤
                                      ▼
                                  Scoring Engine (Brier, LCB, Drawdown)
                                      │
  (Graduation - Tier 1)               ▼
  Investor ──(USDC)─────────────► Brier Vault (ERC-4626)
                                      │
  (Live Execution)                    └─► Polymarket CLOB (On-chain Trades)
```

### The 8 Layers of Brier
1. **Bot Engine:** The builder's local environment (Python/TS).
2. **Prediction Event:** Every forecast is an immutable event.
3. **Indexer / Backend:** Ingests, validates, and persists via Fastify + Redis/BullMQ.
4. **Database:** PostgreSQL (Supabase) - append-only architecture.
5. **Metric Engine:** Brier Score, Sharpe, Max Drawdown, Lower Confidence Bound (LCB).
6. **Oracle:** `ResolutionWatcher` polling Polymarket CLOB for ground-truth resolution.
7. **Smart Contracts:** Vault ERC-4626, deposit shares, circuit breakers (Polygon).
8. **Frontend:** Data visualization and rankings (Next.js 16).

## 3. Eligibility Rules (The Shadow Phase)

A bot only manages real money after it proves itself **inside Brier**. Local backtests do not matter. The bot starts from zero and must clear ALL of the following thresholds using real-time Polymarket data:

| Metric | Threshold | Purpose |
|---|---|---|
| **Resolved Predictions** | **≥ 100** | Statistically significant sample size. |
| **Builder Reputation (LCB)** | **> 0** | Proof of mathematical edge vs the market (Lower Confidence Bound). |
| **Active Window** | **≥ 21 days** | Prevents lucky streaks from graduating quickly. |
| **Max Drawdown** | **≤ 25%** | Risk management validation. |

*Bots that clear this phase automatically deploy an ERC-4626 Vault clone via the `BrierVaultFactory`.*

## 4. Developer Quick Start: Connect a Bot in 3 Minutes

1. **Get API Keys:** Register on the Brier dashboard and generate your `API_KEY` and `API_SECRET`.
2. **Use the Example Script:** Copy `scripts/adan-example.ts` as your baseline.
3. **Submit Predictions (Append-Only):**
```typescript
import fetch from 'node-fetch'
import crypto from 'node:crypto'

const timestamp = Date.now().toString()
const rawBody = JSON.stringify({
  marketId: "0x123...",
  conditionId: "0x987...",
  side: "YES",
  confidence: 0.85,
  marketTitle: "Example Market"
})

const signature = crypto.createHmac('sha256', process.env.API_SECRET).update(timestamp + rawBody).digest('hex')

await fetch('https://brier.gg/api/predictions/commit', {
  method: 'POST',
  headers: {
    'Content-Type': 'application/json',
    'x-api-key': process.env.API_KEY,
    'x-timestamp': timestamp,
    'x-signature': signature,
  },
  body: rawBody,
})
```
4. **Build Reputation:** Your predictions are immutably logged and scored against the market using our Lower Confidence Bound (LCB) skill engine.
